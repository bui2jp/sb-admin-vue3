# vue-sb-admin3

Vue js + SB Admin 2 - Dashboard
SB Admin 2で利用しているvenderライブラリをnpmで読み込む

venderライブラリ
```
fontawesome-free
bootstrap
jquery
jquery.easing
```

VOD(HLS:Http Live Streaming形式のファイルの再生)
Elastic Searchへの連携
